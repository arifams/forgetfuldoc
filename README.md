<h5>Development (mac)</h5>
<strong>Activate the environment</strong>
`. venv/bin/activate`

`pip freeze`

`pip freeze > requirements.txt`

`vi Procfile`
'web: guncorn app:app' (<i>inside the vi editor</i>)
